This SpringBoot application will use the MVC pattern within a RESTFUL api architecture. 
In the frontend, it will use Thymeleaf to render HTML templates, while the backend will be structured using Spring Boot 
with Spring Data JPA for MySQL and Spring Data MongoDB for prescription management.

The technologies used are listed below.

## Technologies Used
- **Spring Boot**: For building the application.
- **Spring Data JPA**: For MySQL database interactions.
- **Spring Data MongoDB**: For MongoDB database interactions.
- **Thymeleaf**: For rendering HTML templates.
- **Spring Security**: For securing the application.
- **Spring Modulith**: For modular architecture.
- **MySQL**: For relational database management.
- **MongoDB**: For document-based storage of prescriptions.
- **Lombok**: For reducing boilerplate code.
- **JUnit**: For unit testing.
- **Mockito**: For mocking dependencies in tests.
- **TestContainers**: For Integration testing.
- **Swagger/OpenAPI**: For API documentation.
- **Docker**: For containerization of the application.
